Obs.: tentar inserir com id=3 lançará uma exceção de "Menor de idade inválido".

Inserir com id=5 fará com que seja lançada uma exceção e o objeto não vai persistir no Banco.
Lembre-se que a cada reinício do servidor, os dados serão apagados.
